# Project Instructions: willluft-2026

## Graffiti Best Practices

This project uses the Graffiti CSS design system (`css/graffiti.css`). The `graffiti-best-practices` skill is **always active** for any UI or markup work in this project.

Apply the skill for any task involving:
- Writing or editing HTML markup
- Writing or editing `css/styles.css`
- Adding new pages or sections
- Refactoring existing classes or layout

**Do not modify `css/graffiti.css`** — it is the upstream design system and must remain untouched.

## Class Naming

Use flat hyphenated class names. Do not use BEM (`__` or `--` syntax).
